# K리그 가상 축구단 '성만 FC' 공식 홈페이지 개발 스펙

이 문서는 수원 삼성 블루윙즈의 열정적인 팬 문화를 모티브로 하고, 세련된 '다크 앤 골드' 테마를 적용한 K리그 가상 축구단 **성만 FC (Sungman FC)**의 공식 홈페이지 개발 설계서입니다.

---

## 1. 프로젝트 개요

*   **프로젝트명**: 성만 FC 공식 홈페이지 개발 (Sungman FC Official Website)
*   **구현 모델**: 단일 페이지 애플리케이션 (SPA) 형태의 반응형 웹 사이트
*   **핵심 콘셉트**: 
    *   **Dark & Gold 프리미엄 비주얼**: 어둡고 묵직한 카본 블랙 배경 위에 찬란한 황금빛 포인트를 가미하여 전통 명문 구단의 품격을 시각화.
    *   **몰입감 있는 사용자 경험(UX)**: 부드러운 탭 전환 애니메이션과 선수 카드 모달, 새로고침 없는 상태 변경 등을 적용하여 앱과 같은 완성도 제공.
    *   **실감나는 데이터**: 선수 프로필, 경기 일정, 가상 리그 순위표를 데이터화하여 동적 바인딩.
    *   **양방향 소통 공간**: LocalStorage를 활용한 익명 게시판 및 댓글 시스템 구축.

---

## 2. 기술 스택

*   **HTML5**: 시맨틱 웹 표준 준수 (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>` 등)
*   **CSS3 (Vanilla CSS)**: CSS Custom Properties(테마 변수) 적용, Flexbox/Grid 레이아웃, Glassmorphism 효과, 키프레임 애니메이션
*   **JavaScript (Vanilla JS)**: ES6+ 표준, DOM 이벤트 제어, 탭 라우팅, LocalStorage 데이터 영속화

---

## 3. 디렉터리 및 파일 구조

```text
fcclub/
├── index.html           # 전체 레이아웃 골격 및 공통 헤더/푸터 정의
├── css/
│   ├── main.css         # 전체 레이아웃 스타일
│   ├── theme.css        # 다크 앤 골드 테마 컬러 변수 및 공통 타이포그래피
│   └── components.css   # 카드, 모달, 게시판, 테이블 등 컴포넌트 단위 스타일
├── js/
│   ├── app.js           # 초기화, 탭 라우팅 및 전역 UI 인터랙션 관리

└── assets/              # 구단 로고, 선수 이미지, 매치 경기장 목업 등 리소스 파일
```

## 4. UI/UX 디자인 시스템 (`theme.css`)

### 4.1. 컬러 팔레트 (CSS 변수)
*   `--color-bg-primary`: `#0d0d0d` (깊고 어두운 메인 카본 블랙)
*   `--color-bg-secondary`: `#1a1a1a` (카드 및 컨테이너용 딥 그레이)
*   `--color-accent-gold`: `linear-gradient(135deg, #f39c12, #d4af37, #f9ca24)` (로열 골드 그라데이션)
*   `--color-gold-solid`: `#d4af37` (단색 골드 포인트)
*   `--color-text-primary`: `#ffffff` (주 텍스트)
*   `--color-text-muted`: `#aaaaaa` (부 텍스트 및 메타 정보)
*   `--color-glass-bg`: `rgba(26, 26, 26, 0.7)` (반투명 헤더 배경)
*   `--color-glass-border`: `rgba(212, 175, 55, 0.15)` (미세한 골드 빛 테두리)

### 4.2. 타이포그래피
*   구글 웹폰트 `Outfit` 또는 `Inter` 적용 (현대적이고 고급스러운 산세리프 폰트)
*   헤드라인: `font-family: 'Outfit', sans-serif; font-weight: 700;`
*   본문: `font-family: 'Inter', sans-serif; font-weight: 400;`

## 5. 상세 기능 및 구현 명세 (탭별 구성)

### 5.1. Header & Navigation (공통)
*   스크롤 시 상단에 고정되는 Sticky Header.
*   `backdrop-filter: blur(12px)` 및 미세한 골드 보더를 활용한 글래스모피즘 효과.
*   황금빛 성만 FC 엠블럼과 5개의 메뉴 버튼 (`Home`, `Club`, `Squad`, `Matches`, `Fan Zone`).
*   활성화된 메뉴는 골드 컬러 밑줄 애니메이션으로 강조.

### 5.2. Home (홈) 탭
*   **히어로 배너**: 가상 구단 성만 FC의 상징적 메인 비주얼 슬라이드쇼 또는 단일 와이드 배너. 메인 슬로건: `"WE ARE SUNGMAN - 황금빛 날개로 비상하라"`
*   **D-Day 매치 위젯**: 가장 가까운 경기 일정(상대 팀, 시간, 경기장 정보) 및 남은 시간 카운트다운 타이머.
*   **최근 소식 (News Grid)**: 영입 소식, 구단 공식 발표 등 카드형 그리드 (호버 시 스케일 업 애니메이션 효과).

### 5.3. Club (구단 소개) 탭
*   **엠블럼 정보**: 성만 FC 엠블럼의 상징 요소(수호를 뜻하는 사자, 성장을 뜻하는 날개)를 세련되게 설명.
*   **홈구장 가이드**: '성만 아레나(Sungman Arena)' 구장의 크기, 수용량(40,000석), 오시는 길 등을 매력적인 레이아웃으로 소개.

### 5.4. Squad (선수단) 탭
*   **카테고리 필터**: `전체`, `GK`, `DF`, `MF`, `FW` 버튼 제공. 클릭 시 해당 포지션 선수만 부드러운 Fade-in 효과와 함께 정렬.
*   **선수 프로필 카드**: 
    *   선수 사진, 등번호, 한글/영문 성명, 포지션 표시.
    *   골드 하이라이트 테두리 및 그림자 효과.
*   **선수 상세 정보 모달**:
    *   카드 클릭 시 선수의 이번 시즌 주요 스탯(출장 경기 수, 득점, 도움, 패스 성공률 등) 및 상세 프로필(키, 몸무게, 생년월일)을 팝업 형태로 출력.
    *   바깥쪽 클릭 혹은 `ESC` 키 입력으로 닫기 가능.

### 5.5. Matches (매치 센터) 탭
*   **경기 일정 및 결과 리스트**: 이번 시즌 경기 결과(스코어 포함) 및 잔여 일정을 카드형 목록으로 정렬.
*   **리그 순위표**: K리그 1 주요 구단들과 성만 FC의 가상 순위(성만 FC는 리그 우승 경쟁 중인 2~3위로 노출) 시각화.

### 5.6. Fan Zone (팬 커뮤니티) 탭
*   **실시간 한줄 응원 게시판**: 닉네임과 응원 한 마디를 적어 등록하면, 우측 스크롤 배너에 흘러가는 효과나 최신 순 한줄 보드로 출력.
*   **자유 게시판**:
    *   목록 뷰: 제목, 작성자, 작성일, 조회수 표시.
    *   글쓰기 폼: 닉네임, 비밀번호(삭제용), 제목, 본문 입력 가능.
    *   상세 뷰: 글 내용 조회 및 해당 글 아래에 댓글(닉네임, 내용) 작성 및 등록 기능.
    *   데이터 관리: `localStorage`를 사용하여 페이지 새로고침 시에도 글 및 댓글 보존.

## 6. 데이터 구조 및 흐름

### 6.1. 데이터 모델 (`data.js`)
*   `squadData`: `Array<{ id, name, engName, number, position, image, stats: { matches, goals, assists }, details: { height, weight, birth } }>`
*   `matchData`: `Array<{ id, opponent, date, time, venue, type: 'Home'|'Away', status: 'upcoming'|'finished', score: { home, away } }>`
*   `standingData`: `Array<{ rank, teamName, played, wins, draws, losses, gd, points }>`

### 6.2. 상태 관리 흐름
1.  **탭 변경**: `app.js`에서 네비게이션 링크 클릭 이벤트를 구독하여, 모든 섹션을 감춘 뒤 선택된 섹션만 `display: block` 처리.
2.  **모달 열기**: `Squad` 섹션의 선수 카드 클릭 시, `data-id` 속성으로 선수를 찾아 모달 창 HTML 템플릿에 주입 후 모달 클래스 토글 (`.is-visible`).
3.  **게시글 등록**: `community.js`에서 글 작성 폼 제출 감지 -> 로컬 스토리지 데이터 로드 -> 새 데이터 추가 -> 로컬 스토리지 저장 -> 게시판 목록 리렌더링.

## 7. 검증 및 테스트 시나리오

1.  **반응형 레이아웃 확인**: 모바일, 태블릿, 데스크톱 기기 환경에서 레이아웃이 깨지지 않고 햄버거 메뉴(모바일) 또는 적절한 그리드가 노출되는지 검증.
2.  **탭 내비게이션 검증**: 각 메뉴 클릭 시 스무스하게 탭이 전환되고 스크롤 위치가 최상단으로 복귀하는지 테스트.
3.  **선수단 필터 및 모달 작동**: 포지션 버튼 필터가 정확히 매칭되는지, 카드를 눌렀을 때 해당 선수의 기록 모달이 제대로 표시되고 닫히는지 확인.
4.  **게시판 영속성 검증**: 글 작성 후 새로고침 시 데이터 유지 여부, 댓글 추가 및 올바른 렌더링 확인.
5.  **테마 스타일링 정합성**: 다크 모드 기반의 고대비 로열 골드 컬러가 전체 일관성을 유지하는지 확인.
