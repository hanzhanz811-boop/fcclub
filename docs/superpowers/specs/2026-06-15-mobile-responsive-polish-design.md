# 2026-06-15 모바일 반응형 및 UI 다듬기 디자인 스펙

## 1. 개요
성만 FC Single Page Application(SPA) 웹사이트의 모바일 사용성을 극대화하기 위해 반응형 레이아웃 튜닝 및 프리미엄 모바일 하단 고정 탭바(Bottom Navigation Bar)를 구현하고, 전체적인 웹 UI 디테일을 다듬습니다.

## 2. 모바일 하단 고정 탭바 (Bottom Navigation Bar)
*   **레이아웃 및 위치**: 모바일 화면(`max-width: 768px`)이 활성화될 때 화면 맨 하단에 `position: fixed; bottom: 0; left: 0; width: 100%; z-index: 2000;`으로 고정됩니다.
*   **헤더 연동**:
    *   기존 헤더의 네비게이션 가로 메뉴(`header nav`)는 모바일 화면에서 숨깁니다 (`display: none;`).
    *   모바일 화면 하단에 새로운 탭바 `<nav class="mobile-nav-bar">`를 렌더링합니다.
    *   이 탭바는 `index.html` 하단 또는 적절한 위치에 정적으로 정의되어 데스크톱에서는 숨겨지고(`display: none;`) 모바일에서만 노출됩니다.
*   **아이콘 및 구성**:
    *   총 5개 탭: 홈(Home), 구단(Club), 선수단(Squad), 경기(Matches), 팬존(Fan Zone).
    *   각 탭은 가독성이 높고 직관적인 모던 SVG 아이콘 + 텍스트 라벨로 구성됩니다.
*   **스타일 & 비주얼**:
    *   `background: var(--color-glass-bg);` 및 `backdrop-filter: blur(15px);`를 적용하여 반투명 유리 질감을 줍니다.
    *   활성 탭(`active`)에는 골드 채우기 색상(`fill: var(--color-gold-solid); color: var(--color-gold-solid);`)을 적용하고 마이크로 인터랙션(탭 클릭 시 스케일이 살짝 커지는 바운스 효과)을 가미합니다.

## 3. 섹션별 모바일 반응형 및 UI 폴리싱
*   **홈 섹션 (Home)**:
    *   히어로 배너의 타이틀 폰트 크기 축소 (`font-size: 2.5rem` -> `1.8rem`).
    *   뉴스 카드 리스트 그리드를 `grid-template-columns: 1fr;`로 변경하고 패딩 조절.
*   **구단 정보 (Club)**:
    *   소개 섹션 및 아레나 가이드 레이아웃을 세로 1단으로 스택. 아레나 배경 사진의 높이를 모바일에 맞추어 적절하게 크롭(height: 250px)하여 여백 최적화.
*   **선수단 (Squad)**:
    *   선수 필터 버튼 리스트 가로 스크롤 활성화 (`overflow-x: auto; white-space: nowrap;` 적용)하여 버튼이 아래로 떨어져서 지저분해지는 현상 방지.
    *   선수 카드 그리드를 모바일에서는 2열(`grid-template-columns: repeat(2, 1fr);`) 또는 1열로 튜닝.
    *   상세 스탯 모달의 모바일 너비(90% 너비) 확장 및 패딩 슬림화.
*   **매치 센터 (Matches)**:
    *   일정 리스트와 실시간 순위표를 세로 1열로 변경.
    *   순위표 테이블에 가로 스크롤(`overflow-x: auto;`) 속성을 감싸서 작은 화면에서도 레이아웃 깨짐을 방지.
*   **팬존 (Fan Zone)**:
    *   2단 레이아웃을 세로 1열 스택으로 전환.
    *   게시글 리스트 하단 마진을 추가하여 탭바에 가려지지 않게 설정.
    *   게시글 상세 정보 조회 또는 글쓰기 양식 오픈 시 모바일 환경에서는 해당 스크롤이 자동으로 우측 컬럼(세로 스택 시에는 아래 컬럼)으로 부드럽게 스크롤되도록 인터랙션 스크립트 결합.

## 4. 종합 테스트 및 무결성 검증
*   `tests/run_tests.js`를 재실행하여 신규 모바일 마크업 추가 후에도 기존 데이터 모델 및 라우터 기능이 영향 없이 안전하게 100% PASS되는지 확인.
*   크롬 개발자 도구를 통한 모바일 시뮬레이터(iPhone SE, iPhone 12 Pro, iPad 등) 반응형 뷰포트 레이아웃 깨짐 및 하단 고정바 사용성 검증.
