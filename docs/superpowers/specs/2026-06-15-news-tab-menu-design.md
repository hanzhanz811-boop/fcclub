# 2026-06-15 뉴스(News) 전용 메뉴 신설 디자인 스펙

## 1. 개요
기존 모달창 방식 대신 독립된 **뉴스(News) 탭** 메뉴를 신설하고, 홈 탭의 뉴스 기사 클릭 시 해당 뉴스 탭으로 이동하여 해당 기사 본문이 활성화되도록 개편합니다.

## 2. 네비게이션 구조 변경
*   **데스크톱 헤더 네비게이션 (`index.html`)**:
    *   Home, Club, Squad, Matches, **News**, Fan Zone 순서로 메뉴를 배치합니다.
*   **모바일 하단 고정 탭바 (`index.html`)**:
    *   총 6개 탭으로 구성하거나 가로 공간 확보를 위해 아래의 6개 탭으로 자연스럽게 배열합니다:
    *   홈(Home), 구단(Club), 선수단(Squad), 경기(Matches), **뉴스(News)**, 팬존(Fan Zone)

## 3. 뉴스 탭 레이아웃 및 스타일 정의
*   **HTML 구조 (`index.html`)**:
    *   `#news` 섹션을 추가하고 팬존과 유사한 2단 레이아웃(왼쪽: 뉴스 목록, 오른쪽: 선택 기사 상세 본문)을 구축합니다.
*   **CSS 스타일 (`css/components.css`)**:
    *   팬존의 레이아웃 스타일을 그대로 활용하거나 클래스명을 분리해 가독성을 높입니다.
    *   모바일 뷰포트 시 세로 1단 배치 및 여백 조절을 적용합니다.

## 4. 라우팅 및 인터랙션 엔진 업데이트 (`js/app.js`, `js/community.js`)
*   **홈화면 클릭 연동**:
    *   홈 탭에서 뉴스 타이틀 클릭 시 전역 상태 `activeNewsId`에 해당 뉴스 ID를 세팅하고, `window.location.hash = 'news'`로 해시 값을 변경하여 탭 전환을 유도합니다.
*   **뉴스 탭 렌더링 (`window.renderNewsPage`)**:
    *   `js/app.js` 내에 뉴스 전용 리스트 및 상세 본문 렌더링 로직을 결합합니다.
    *   해당 탭이 활성화될 때 왼쪽 뉴스 목록(활성화된 카드 골드 배경 표시)과 우측 기사 상세 내용이 동시에 로드됩니다.
    *   모바일 환경에서는 뉴스 리스트 카드 클릭 시 하단 상세 페이지로 자동 부드러운 스크롤(`scrollIntoView`) 연동을 추가합니다.
